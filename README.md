# Scan Viewer

Three.js teeth scan viewer with local ZIP upload support.

This repository intentionally does not include patient scan assets. ZIP files are unpacked locally in the browser and are not uploaded to a server.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The app is configured for GitHub Pages at the custom domain root:

https://scan.seaty6.com/
